üìã How to Add Notifications to Any Page

Step-by-Step Process:

Step 1: Include the notification HTML structure in your page
Step 2: Add the JavaScript notification functions
Step 3: Call addNotification() when something happens
Step 4: The notification automatically appears in the bell dropdown

üíª Code Implementation

1. JavaScript Function to Add Notifications:

function addNotification(type, title, message) { const notificationList = document.getElementById('notification-list'); const emptyNotifications = document.getElementById('empty-notifications'); const notificationBadge = document.getElementById('notification-badge'); // Hide empty state if (emptyNotifications) { emptyNotifications.style.display = 'none'; } // Create notification element const notification = document.createElement('div'); notification.className = 'notification-item unread'; notification.innerHTML = ` <div class="notification-content"> <div class="notification-icon ${type}"> ${getNotificationIcon(type)} </div> <div class="notification-text"> <div class="notification-title">${title}</div> <div class="notification-message">${message}</div> <div class="notification-time">${getTimeAgo(new Date())}</div> </div> </div> `; // Add to list notificationList.appendChild(notification); // Update badge count updateNotificationBadge(); // Show toast notification showToastNotification(type, title, message); }

2. How to Trigger Notifications:

// Example: When user books an appointment function bookAppointment() { // Your booking logic here... // Add notification addNotification( 'booking', // Type 'Booking Confirmed', // Title 'Your appointment is scheduled for tomorrow at 2:00 PM' // Message ); } // Example: When user completes a skin test function completeSkinTest() { // Your test completion logic... addNotification( 'update', 'Test Complete', 'Your skin analysis results are ready to view!' ); }

3. Notification Types Available:

// Available notification types: 'booking' - Green icon, for appointments/bookings 'reminder' - Yellow icon, for reminders and alerts 'update' - Blue icon, for updates and information 'clinic-update' - Purple icon, for clinic-related news
